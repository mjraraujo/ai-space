# AI Space — Product Vision

## One-line vision
AI Space is a privacy-first personal AI environment that uses the smartphone as the trusted execution surface.

## Core concept
AI Space is not only a chat assistant. It is a control plane for the user's digital life.

It should:
- understand user intent
- observe allowed context from the device
- plan actions
- execute safe actions
- request approval for risky ones
- keep privacy and trust visible

## What makes it different
The device is the trust anchor.

Instead of asking the user to reconnect every service through raw API keys, AI Space uses:
- the phone's permission model
- the phone's current authenticated app state
- on-device biometrics
- local models where possible

## User promise
- Private: no training on user data by default
- Personal: can remain tool-like or become assistant-like
- Powerful: coordinates work across apps and routines

## Why users would switch
Not because it replaces the phone.
Because it makes the phone feel like a coherent environment instead of a bag of disconnected apps.

## Best first wedge
Android-first, local-first AI layer for:
- message summarization
- task extraction
- reply drafting
- approval-based guided actions
